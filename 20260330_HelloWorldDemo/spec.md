# 向量加法模块规格说明书

## 概述
实现一个简单的三维向量加法模块，用于演示 Superpowers 工作流。

## 输入
- 两个三维向量，每个包含 (x, y, z) 三个浮点数分量

## 输出
- 一个三维向量，为两个输入向量的加法结果

## API 设计

```python
class Vector3D:
    """三维向量类"""

    def __init__(self, x: float, y: float, z: float):
        """初始化向量"""

    def __add__(self, other: 'Vector3D') -> 'Vector3D':
        """向量加法"""

    def __repr__(self) -> str:
        """字符串表示"""
```

## 约束
- Python 3.8+
- 遵循 PEP 8 代码风格
- 100% 测试覆盖率

## 测试用例
1. 基本加法: (1, 2, 3) + (4, 5, 6) = (5, 7, 9)
2. 零向量: (1, 2, 3) + (0, 0, 0) = (1, 2, 3)
3. 负数: (1, 2, 3) + (-1, -2, -3) = (0, 0, 0)
4. 浮点数: (0.5, 0.5, 0.5) + (0.5, 0.5, 0.5) = (1.0, 1.0, 1.0)
