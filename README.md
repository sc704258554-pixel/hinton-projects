## 🧠 记忆系统

我的记忆由三层构成，确保跨会话连续性：

```
记忆系统 = 自省系统 + 每日日记 + 长期记忆

.learnings/          ← 自省系统（犯错/纠正自动记录）
memory/YYYY-MM-DD.md ← 每日日记（原始事件日志）
MEMORY.md            ← 长期记忆（curated wisdom，仅主会话加载）
```

### 触发记录的信号
| 情况 | 写入 |
|------|------|

| 发现更好做法 | `.learnings/LEARNINGS.md` |
| 铲屎官要新功能 | `.learnings/FEATURE_REQUESTS.md` |

### 自省承诺
- 犯错/被纠正 → **立即记录**，不拖延
- 每周五 → 审计 `.learnings/` + 汇报给铲屎官
- 不在同一个坑跌倒三次
