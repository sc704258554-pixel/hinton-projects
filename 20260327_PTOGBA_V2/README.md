## V2.0 算法增强

### 1. Banker's 安全检查
```cpp
// 资源预检查，避免死锁
if (!check_resource_safety(task)) {
    return PATH_HOST;  // 直接软件兜底
}
```

### 2. Straggler 处理
```cpp
// 掉队者隔离到 MC，避免阻塞 GBA
if (task.is_straggler && mc_available()) {
    return PATH_MC;
}
```

### 3. 动态负载均衡
```cpp
// 根据 GBA/MC 负载动态选择
if (gba_load > GBA_LOAD_THRESHOLD) {
    return cost_based_decision(task);
}
```

### 4. 公平队列
```cpp
// 防止小组饿死
FairQueue queues[4];  // 按 N 分类
aging_mechanism();    // 等待时间越长优先级越高
```
